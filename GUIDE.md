# 🍅 Chrono Focus - 使用指南

## 🎯 快速开始

### 开发模式运行
```bash
cd /Users/z/code/VibeCoding/chrono-focus
pnpm dev
```
然后在浏览器打开 **http://localhost:5173**

### 生产模式构建
```bash
pnpm build
```
构建产物在 `dist/` 目录

## 📱 功能说明

### 1. 番茄钟（主界面）
- **🍅 番茄钟**：25分钟专注时间（可自定义）
- **☕ 短休息**：5分钟休息时间
- **🌟 长休息**：15分钟休息时间

**使用方法：**
1. 选择定时器类型（番茄钟/短休息/长休息）
2. 点击"开始"按钮启动计时器
3. 可以随时暂停或重置

### 2. 喝水记录
- 记录每日喝水杯数
- 设定每日喝水目标（默认8杯）
- 可视化进度条显示

**使用方法：**
- 每次喝完一杯水后，点击"记录一杯水"按钮
- 查看今日喝水进度

### 3. 统计面板
查看今日工作统计：
- 完成的番茄钟数量
- 总专注时间（分钟）
- 喝水进度

### 4. 设置面板
可自定义以下参数：

**番茄钟设置：**
- 番茄钟时长（1-60分钟）
- 短休息时长（1-30分钟）
- 长休息时长（1-60分钟）

**休息提醒：**
- 开启/关闭休息提醒
- 提醒间隔（5-120分钟）

**喝水提醒：**
- 开启/关闭喝水提醒
- 提醒间隔（5-120分钟）
- 每日喝水目标（1-20杯）

## 🔔 通知系统

应用会使用系统通知提醒你：
- 番茄钟完成
- 休息时间结束
- 定时休息提醒
- 喝水提醒

首次使用时浏览器会请求通知权限，请允许。

## 💡 使用技巧

1. **高效工作法**
   - 完成4个番茄钟后，进行一次长休息
   - 每30分钟起身活动一下
   - 保持每小时喝一杯水

2. **自定义设置**
   - 根据个人习惯调整时间
   - 可以关闭不需要的提醒

3. **数据持久化**
   - 所有设置自动保存到浏览器
   - 下次打开时自动恢复

## 🐛 常见问题

**Q: 通知没有弹出？**
A: 请检查浏览器通知权限设置，确保允许了通知。

**Q: 设置没有保存？**
A: 设置会自动保存到浏览器的 localStorage。如果清除了浏览器数据，设置会丢失。

**Q: 如何重置所有设置？**
A: 清除浏览器的 localStorage 即可。
